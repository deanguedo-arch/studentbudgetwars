# Intended Weekly Loop

The long-term target loop is a simple weekly cadence:

1. Show current week summary and player status.
2. Surface bills, recurring expenses, and location context.
3. Let the player choose actions such as work, study, rest, or managing resources.
4. Resolve job income, expenses, and budget effects.
5. Roll random life events.
6. Update stress, energy, debt, savings, and score.
7. Check for end-of-term success or failure.

This first pass does not implement the full loop yet. It only creates the shell needed to support that loop cleanly in later passes.
